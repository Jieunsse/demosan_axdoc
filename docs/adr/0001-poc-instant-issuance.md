# ADR 0001: PoC는 즉시 자동 발급 방식으로 구현한다

## 상태
채택됨

## 맥락
재직증명서 발급 플로우를 설계할 때 "즉시 자동 발급"과 "관리자 승인 후 발급" 두 방식을 검토했다.

## 결정
PoC는 즉시 자동 발급으로 구현한다. 승인 플로우는 추후 옵션으로 추가할 수 있도록 설계 여지를 남긴다.

## 근거
- 이 서비스의 핵심 가치는 "HR 담당자 없이 즉시 발급"이다. 승인을 넣으면 기존 프로세스와 차별화가 사라진다.
- 승인이 필요한 회사는 소수이며, PoC 검증 단계에서 다수의 페인포인트를 먼저 해결하는 것이 우선이다.

## 결과
- 승인 발급이 필요한 회사는 PoC 단계에서 대응 불가. 추후 기능으로 대응 예정.
