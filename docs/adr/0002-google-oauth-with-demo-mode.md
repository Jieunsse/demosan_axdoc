# ADR 0002: 인증은 Google OAuth, PoC 검증용 둘러보기 모드를 제공한다

## 상태
채택됨

## 맥락
직원 접근 방식으로 이메일+비밀번호, AXhub 계정 통합, Google OAuth 등을 검토했다. 또한 PoC 검증을 위해 로그인 없이 서비스를 체험할 수 있는 방법이 필요했다.

## 결정
- 인증: Google OAuth (회사용 구글 계정)
- PoC 검증: 둘러보기 모드 (로그인 없이 흐름 체험 가능)

## 근거
- 국내 사무직 회사 대부분이 Google Workspace를 사용한다.
- AXhub 내부 구조가 미확정 상태이므로 AXhub 계정 통합은 블로커가 된다.
- 둘러보기 모드는 고객사에 PoC를 시연할 때 계정 없이 가치를 빠르게 증명할 수 있다.

## 결과
- Google Workspace를 사용하지 않는 회사는 PoC 단계에서 대응 불가.
- AXhub 계정 통합은 내부 구조 확정 후 추가 예정.
